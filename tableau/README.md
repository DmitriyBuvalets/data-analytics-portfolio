# Sales Dashboard

## Overview

Interactive Tableau dashboard built using the Tableau Sample Superstore dataset.

## Business Goal

Analyze sales performance, profit, customer segments, and regional trends.

## Dashboard Features

- KPI overview
- Sales by Region
- Sales by Category
- Monthly Sales Trend
- Interactive Filters

## Files

- Sales_Dashboard.twb – Tableau workbook
- Sales_Dashboard.pdf – Dashboard preview
- Sample - Superstore.xls – Sample dataset

## Technologies

- Tableau
- SQL
- Excel
